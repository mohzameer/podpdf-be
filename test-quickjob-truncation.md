## Expected Response

**Status Code:** `200 OK`

**Headers:**
- `Content-Type: application/pdf`
- `X-PDF-Pages: 2` (should show 2, not 3)
- `X-PDF-Truncated: true` (should be `true`)
- `X-Job-Id: <uuid>`

**Body:** Binary PDF content (2 pages only)

## Testing Notes

1. **Replace placeholders:**
   - `YOUR_API_URL` - Your API Gateway URL (e.g., `https://abc123.execute-api.eu-central-1.amazonaws.com`)
   - `YOUR_JWT_TOKEN` - A valid JWT token from Cognito

2. **Verify truncation:**
   - Check the `X-PDF-Truncated` header is `true`
   - Check the `X-PDF-Pages` header is `2`
   - Open the PDF and verify it only contains 2 pages
   - Verify Page 3 content is not present

3. **Save PDF to file:**
   ```bash
   curl -X POST https://YOUR_API_URL/quickjob \
     -H "Content-Type: application/json" \
     -H "Authorization: Bearer YOUR_JWT_TOKEN" \
     -d @request.json \
     --output test-output.pdf
   ```
